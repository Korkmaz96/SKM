# SKM Management – Webseite

## Technologie

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui (Radix UI)
- React Router v6

## Lokale Entwicklung

```sh
npm install
npm run dev
```

Dev-Server startet auf http://localhost:8080

## Build

```sh
npm run build
```

Output liegt in `/docs` (GitHub Pages).

## Deployment

Die Seite wird über GitHub Pages aus dem Branch `feat/skm-controlled-setup` (Ordner `/docs`) ausgeliefert.

Domain: skm-management.de
